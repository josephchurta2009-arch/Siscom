# 📘 Manual de Usuario – Sistema CISCOM

## 1. Introducción
El sistema **CISCOM** es una plataforma diseñada para gestionar información comunitaria, incluyendo registro de personas, ayudas y entregas.

---

## 2. Acceso al sistema
1. Abrir el navegador web  
2. Ingresar la URL del sistema  
3. Iniciar sesión con:
   - Usuario
   - Contraseña  

---

## 3. Módulos principales

### 👤 Gestión de Personas
Permite registrar y administrar usuarios beneficiarios.

**Funciones:**
- Registrar persona
- Editar datos
- Eliminar registros
- Consultar información

---

### 🎁 Gestión de Ayudas
Permite crear y administrar tipos de ayuda.

**Funciones:**
- Registrar ayuda
- Editar ayuda
- Eliminar ayuda

---

### 📦 Gestión de Entregas
Controla la asignación de ayudas a personas.

**Funciones:**
- Registrar entrega
- Consultar historial
- Validar entregas realizadas

---

## 4. Flujo básico de uso

1. Registrar persona  
2. Crear ayuda  
3. Asignar ayuda (entrega)  
4. Consultar reportes  

---

## 5. Recomendaciones
- Verificar datos antes de guardar  
- No duplicar registros  
- Cerrar sesión al terminar  

---

## 6. Soporte
En caso de errores, contactar al administrador del sistema.
